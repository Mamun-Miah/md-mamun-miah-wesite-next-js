# Next.js Headless Blog (WordPress)

This is a **Next.js App Router** project bootstrapped with  
[`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

The project demonstrates how to build a **headless blog frontend** using **Next.js** while fetching **WordPress single post data** from **mamundev.com** via the WordPress REST API.
